# flight-controller
stm32f405 based flight controller.
my project is a flight controller designed for quadcopter fpv drones
this flight controller can be used by first ordering the pcb and electonics, putting them together and flashing betaflight on to the stm32f405 using a custom target
i made this project because i wanted to improve my pcb and circuit design skill. Not to mention that i love fpv drones and i wanted something that i could fly around my house when i was bored. so that's why i made it
